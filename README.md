## 계좌 단위

- 미국 직접투자 전략은 `USD` 기준으로 평가합니다
- 미국 직접투자 전략에는 환율 변환을 적용하지 않습니다
- ISA는 `KRW` 기준을 유지합니다
- 미국 직접투자 월 적립금을 사용하는 경우, 사이클 온 적립금은 `SPYM`, 리스크 오프 적립금은 `SGOV`를 매수하며 `SGOV` 상장 전에는 `BIL`을 대체로 사용합니다

### `us-soxl-6step (Loan-aware)`

- 자산: `SOXL`
- 진입: 미국 거래일 기준 `close > SMA200`이 3일 연속 성립

  - `+45%` 수익 시: 남은 수량의 `10%` 매도 (첫 번째 확률 고원 활용)

  - `+210%` 수익 시: 남은 수량의 `50%` 매도 (보너스 수익 확정)
  - `+250%` 수익 시: 전량 매도 (사이클 종료)
