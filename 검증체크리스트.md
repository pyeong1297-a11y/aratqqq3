# 전략 검증 체크리스트

## 목적

이 문서는 백테스트 결과를 볼 때 놓치기 쉬운 취약점을 점검하기 위한 체크리스트다.
숫자가 좋아 보여도 아래 항목을 통과하지 못하면 실전 신뢰도는 낮게 본다.

## 1. 데이터 검증

- `TQQQ`, `BULZ`, `SPYM`, `SGOV`, `QQQ`, `KODEX` 데이터가 모두 정상적으로 정렬되어 있는가
- 분할/병합 반영 가격인지 확인했는가
- 결측일, 휴장일, 한국/미국 거래일 불일치를 처리했는가
- `TQQQ vs BULZ` 비교는 공통 구간으로 잘랐는가
- `BULZ`는 실제 거래 가능한 기간과 가상 데이터 구간을 구분했는가

## 2. 신호 검증

- `종가 > SMA200`, `종가 < SMA200`, `종가 = SMA200` 처리 규칙이 명확한가
- `TQQQ 3일`, `BULZ 2일`, `ISA 1일` 확인일 규칙이 코드에 정확히 반영되었는가
- 같은 사이클에서 익절 트리거가 중복 발동하지 않는가
- 청산일에 익절이 동시에 처리되지 않는가

## 3. 체결 검증

- 미국 전략은 `당일 종가 신호 + 당일 애프터장 체결`로 계산되는가
- 미국 전략에서 체결 기준가는 `당일 종가`이고, 차이는 슬리피지로만 처리되는가
- ISA는 `미국 종가 신호 -> 한국 다음 거래일 체결`로 계산되는가
- ISA에서 같은 날 한국 종가 체결 같은 룩어헤드가 없는가
- ISA 낙관적/기준/보수적 3시나리오가 모두 분리되어 계산되는가

## 4. 자금 이동 검증

- 미국 전략 비보유 기간 자금이 `SGOV`로 정상 이동하는가
- 미국 전략 익절금이 `SPYM`으로 정상 이동하는가
- 200일선 이탈 시 `레버리지 + SPYM` 전량이 모두 청산되는가
- ISA 비보유 기간 자금이 `연 3.5% 고정 이자` 가정으로 정상 복리 계산되는가

## 5. 비용 검증

- 미국 전략 수수료 `0.25%`가 매수/매도 모두 반영되는가
- 미국 전략 슬리피지 시나리오 `0.00 / 0.05 / 0.10 / 0.20%`가 모두 계산되는가
- ISA 보수적 시나리오에서 `시초가 ±0.5%` 불리한 체결이 반영되는가
- 실제 가격 데이터를 쓰면서 운용보수를 이중 차감하지 않았는가

## 6. 결과 해석 검증

- `CAGR`만 보지 않고 `MDD`, `총수익률`, `거래횟수`, `보유기간`, `노출비중`도 함께 보는가
- `TQQQ 전체 기간`과 `TQQQ vs BULZ 공통 구간`을 구분해서 보는가
- 슬리피지 변화에 따라 성과가 급격히 무너지는지 확인했는가
- 특정 몇 번의 대박 거래에 성과가 과도하게 의존하지 않는가

## 7. 실전 투입 전 최소 확인

- 기준 시나리오뿐 아니라 보수적 시나리오에서도 전략이 살아남는가
- `익절 없음` 버전과 비교했을 때 익절 규칙이 실제로 개선을 주는가
- `QQQ 200일선`, `TQQQ 단순 보유` 같은 단순 비교군보다 충분히 나은가
- ISA 전략 결과가 공정가치 시나리오에서만 좋아 보이고 실제 시초가 시나리오에서는 급격히 나빠지지 않는가
- `BULZ`는 연구용과 실전용 해석을 분리했는가

## 8. 최종 판단 기준

아래를 모두 만족하면 실전 후보로 본다.

- 공통 구간과 전체 구간 모두에서 구조적으로 일관된 성과가 나온다
- 보수적 체결 가정에서도 MDD와 CAGR이 감당 가능한 수준이다
- 익절, SGOV, SPYM, ISA 체결 시차 규칙이 결과 왜곡 없이 구현되었다
- 특정 자산 하나의 짧은 행운 구간에만 의존하지 않는다

하나라도 크게 어긋나면 규칙을 다시 손봐야 한다.
